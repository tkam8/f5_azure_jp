F5 NetworksのGitHubリポジトリの概要
================================================

Githubは、テンプレートと関連リソースが存在するソース管理システムです。

リポジトリにアクセスするために、以下のリンクをクリックしてください。
<https://github.com/F5Networks/f5-azure-arm-templates>

商用版のテンプレートは、**suppported**というフォルダ配下にあります。
